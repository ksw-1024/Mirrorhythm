-------------------------------------------------------------
	カメラでポストプロセッシングを有効にする方法
-------------------------------------------------------------

1. Package Managerでポストプロセッシングアセットをダウンロードしてください。

2. PostProcessingという名前の新しいレイヤーを作成してください。

3. メインカメラにPost Processing Layerという新しいコンポーネントを追加し、Triggerでメインカメラを選択し、作成したレイヤー（Post Processing Layer）を選択してください。

4. Hierarchy内に新しい空のオブジェクトを作成してください（PP Volumeと呼びます）。

5. その中にPost Processing Volumeという新しいコンポーネントを追加し、「is global」を有効にして、profile-> newをクリックしてください。

6. 「add effect」をクリックして「Ambient Occlusion」と「Bloom」を追加してください。AOの中で「Intensity」を1に設定し、「Bloom」の中でSoft Knee、Clamp、Difussion、そしてIntensityを5に設定して有効にしてください。


注意：エクスポート時にアセットが動作しない場合は、すべてを削除してステップ1から再度開始してください。
